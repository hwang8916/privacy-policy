Privacy Policy

Sobi-kong - Personal Budget App

Last Updated: January 26, 2025

This Privacy Policy explains how Sobi-kong (hereinafter "the App") collects, uses, and protects your personal information. By using the App, you agree to this policy.

1. Information We Collect

1.1 Information You Provide

Type Data Collected Purpose
Account Information Email address, display name, profile photo URL User identification and service provision
Financial Data Transactions, budgets, categories, payment methods Core budget management features
Preferences Language settings, app preferences Personalized user experience

1.2 Automatically Collected Information

  • Usage Statistics: Feature usage frequency, screen navigation (Firebase Analytics)
  • Error Information: App crashes and error logs (Firebase Crashlytics)
  • Device Information: Device type, OS version, app version

2. App Permissions

Permission Purpose Required
Internet (INTERNET) Cloud sync, login, backup/restore Yes
Microphone (RECORD_AUDIO) Voice input for transaction entry No
Note: Microphone permission is only requested when using voice input. You can deny this permission and still use all other app features normally.

3. How We Use Your Information

  • Providing and improving budget management services
  • Synchronizing data across multiple devices
  • User authentication and account management
  • Analyzing errors to improve app stability
  • Analyzing usage statistics to enhance service quality
  • Managing premium subscriptions

4. Third-Party Services

The App uses the following third-party services:

4.1 Firebase (Google)

  • Firebase Authentication: User authentication and login management
  • Cloud Firestore: Cloud data storage and synchronization
  • Firebase Analytics: App usage analytics
  • Firebase Crashlytics: Error tracking and analysis

Firebase Privacy Policy: https://firebase.google.com/support/privacy

4.2 Social Login

  • Google Sign-In: Google Privacy Policy
  • Apple Sign-In: Apple Privacy Policy

4.3 In-App Purchases

Premium subscriptions are processed through Google Play or Apple App Store. Payment information is managed by the respective platform.

5. Data Storage and Security

5.1 Data Storage Location

  • Local Storage: App data is securely stored on your device.
  • Cloud Storage: Logged-in users' data is encrypted and stored on Firebase servers (Google Cloud Platform).

5.2 Security Measures

  • All network communications are encrypted using HTTPS/TLS.
  • Firebase security rules ensure users can only access their own data.
  • Passwords are stored using industry-standard encryption.

6. Data Retention

  • Active Accounts: Data is retained while the account is active
  • Account Deletion: Data is completely deleted within 30 days of deletion request
  • Backup Data: Automatically deleted according to user-configured retention period

7. Your Rights

You have the following rights:

  • Right to Access: Request access to your personal data
  • Right to Rectification: Request correction of inaccurate information
  • Right to Erasure: Request deletion of your personal data
  • Right to Portability: Request data export
  • Right to Withdraw: Withdraw consent for data processing
How to Delete Your Data:
  1. In-app: Settings → Account → Delete Account
  2. Or contact us using the information below

8. Children's Privacy

This App does not knowingly collect personal information from children under 13. If we become aware that a child under 13 has provided personal information, we will delete it immediately.

9. Changes to This Policy

This policy may be updated due to legal requirements or service changes. Significant changes will be communicated through in-app notifications or email.

10. Advertising

This App currently does not display advertisements and does not share personal information with third parties for advertising purposes.
