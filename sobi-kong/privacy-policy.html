개인정보 처리방침

소비콩 (Sobi-kong) - 개인 가계부 앱

최종 업데이트: 2025년 1월 26일

본 개인정보 처리방침은 소비콩(이하 "앱")이 사용자의 개인정보를 어떻게 수집, 사용, 보호하는지 설명합니다. 앱을 사용함으로써 본 방침에 동의하게 됩니다.

1. 수집하는 개인정보

1.1 사용자가 제공하는 정보

정보 유형 수집 항목 수집 목적
계정 정보 이메일 주소, 표시 이름, 프로필 사진 URL 회원 식별 및 서비스 제공
재정 데이터 거래 내역, 예산, 카테고리, 결제 수단 가계부 서비스 핵심 기능 제공
설정 정보 언어 설정, 앱 환경설정 사용자 맞춤 서비스 제공

1.2 자동으로 수집되는 정보

  • 앱 사용 통계: 기능 사용 빈도, 화면 전환 정보 (Firebase Analytics)
  • 오류 정보: 앱 충돌 및 오류 로그 (Firebase Crashlytics)
  • 기기 정보: 기기 유형, 운영체제 버전, 앱 버전

2. 앱 권한 사용

권한 사용 목적 필수 여부
인터넷 (INTERNET) 클라우드 동기화, 로그인, 백업/복원 필수
마이크 (RECORD_AUDIO) 음성 입력으로 거래 등록 선택
참고: 마이크 권한은 음성 입력 기능 사용 시에만 요청되며, 거부하셔도 앱의 다른 기능은 정상적으로 사용할 수 있습니다.

3. 개인정보의 이용 목적

  • 가계부 서비스 제공 및 기능 개선
  • 다중 기기 간 데이터 동기화
  • 사용자 인증 및 계정 관리
  • 앱 안정성 향상을 위한 오류 분석
  • 서비스 품질 향상을 위한 사용 통계 분석
  • 프리미엄 구독 관리

4. 제3자 서비스

앱은 다음의 제3자 서비스를 사용합니다:

4.1 Firebase (Google)

  • Firebase Authentication: 사용자 인증 및 로그인 관리
  • Cloud Firestore: 클라우드 데이터 저장 및 동기화
  • Firebase Analytics: 앱 사용 통계 분석
  • Firebase Crashlytics: 앱 오류 추적 및 분석

Firebase 개인정보처리방침: https://firebase.google.com/support/privacy

4.2 소셜 로그인

  • Google 로그인: Google 개인정보처리방침
  • Apple 로그인: Apple 개인정보처리방침

4.3 인앱 결제

프리미엄 구독은 Google Play 또는 Apple App Store를 통해 처리되며, 결제 정보는 해당 플랫폼에서 관리합니다.

5. 데이터 저장 및 보안

5.1 데이터 저장 위치

  • 로컬 저장소: 앱 데이터는 기기 내 안전하게 저장됩니다.
  • 클라우드 저장소: 로그인 사용자의 데이터는 Firebase 서버(Google Cloud Platform)에 암호화되어 저장됩니다.

5.2 보안 조치

  • 모든 네트워크 통신은 HTTPS/TLS로 암호화됩니다.
  • Firebase의 보안 규칙을 통해 사용자 본인의 데이터만 접근 가능합니다.
  • 비밀번호는 업계 표준 암호화 방식으로 저장됩니다.

6. 데이터 보관 기간

  • 활성 계정: 서비스 이용 기간 동안 보관
  • 계정 삭제 시: 삭제 요청 후 30일 이내 완전 삭제
  • 백업 데이터: 사용자가 설정한 보관 기간에 따라 자동 삭제

7. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 접근권: 본인의 개인정보 열람 요청
  • 정정권: 부정확한 정보 수정 요청
  • 삭제권: 개인정보 삭제 요청
  • 이동권: 데이터 내보내기 요청
  • 철회권: 개인정보 처리 동의 철회
데이터 삭제 방법:
  1. 앱 내 설정 → 계정 → 계정 삭제
  2. 또는 아래 연락처로 삭제 요청

8. 아동 개인정보 보호

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 아동이 개인정보를 제공한 사실을 알게 된 경우, 해당 정보를 즉시 삭제합니다.

9. 개인정보 처리방침 변경

본 방침은 법률 또는 서비스 변경에 따라 수정될 수 있습니다. 중요한 변경 사항은 앱 내 공지 또는 이메일을 통해 안내드립니다.

10. 광고

본 앱은 현재 광고를 표시하지 않으며, 광고 목적으로 개인정보를 제3자와 공유하지 않습니다.
